# Card

Base night-surface card. `accent` adds a top bar (`aurora`/`insight`/`intelligence`/`illumination`); `glow` renders it as an ambient shadow; `glass` = frosted blur; `interactive` = hover-lift.

```jsx
<Card accent="insight" glow>…</Card>
<Card glass interactive padding="var(--space-5)">…</Card>
```
