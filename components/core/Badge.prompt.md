# Badge

Small status / category label. `tone` maps the three I's (`insight` / `intelligence` / `illumination`) plus `neutral` / `success` / `warning` / `danger`. `variant`: `soft` (default) / `solid` / `outline`.

```jsx
<Badge tone="insight" dot>洞察 Insight</Badge>
<Badge tone="success" variant="solid">已發送</Badge>
```
