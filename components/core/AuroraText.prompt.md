# AuroraText

Aurora gradient-clipped display text — masthead, hero numbers, the I³ wordmark. Set `as` for the element tag and `gradient` to override the fill.

```jsx
<AuroraText as="h1" style={{fontSize:"var(--text-5xl)"}}>艾倫報報</AuroraText>
<AuroraText gradient="var(--illumination-gradient)">啟發</AuroraText>
```
